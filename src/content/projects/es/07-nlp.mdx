Analisis de ofertas de empleo tecnologico en Espana mediante web scraping y procesamiento del lenguaje natural (NLP). El proyecto gestiona descripciones de puestos en espanol e ingles, incluye scripts Python para la extraccion automatizada de datos y un notebook interactivo en Google Colab para el analisis exploratorio de datos (EDA), extraccion de palabras clave y modelado de topicos.
