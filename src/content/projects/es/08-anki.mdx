Herramienta CLI en Python impulsada por modelos de lenguaje de gran escala mediante LangChain y la API de OpenAI. Genera flashcards de Anki a partir de cualquier PDF extrayendo el texto, creando pares pregunta-respuesta y exportando a CSV/JSONL para una importacion sencilla en Anki.
